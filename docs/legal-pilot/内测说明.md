# 法务内测版 — 同事入门说明

**版本：** 法务内测版 v1 · 2026-06  
**入口：** http://localhost:3000/review/（用户端） · http://localhost:3000/admin-ui/（管理端）  
**适用对象：** 法务 / 合规同事（中文界面，广告示例文案为英文 SG 场景）

---

## 这是什么？

Ad Compliance Hub（原 AAIRP 工程代号）把 **英文广告文案** 跑过一条 **可审计、可复现** 的审核流水线：

`法规 → 规则 → Playbook →（可选 LLM）→ 决策 → 报告 → 案例库`

本内测版用于验证：**规则 + 法规引用** 路径是否值得法务同事信任，以及 UI 是否便于理解审核逻辑。

---

## 你需要做什么（约 30 分钟）

1. 阅读 [内测范围.md](./内测范围.md)（5 分钟）
2. 确认页面右上角 **API ok**（绿色）
3. 依次运行左侧 **5 个演示场景**，点击 **运行 AI 审核**
4. 每个场景完成后查看：
   - **审核报告** 标签：PASS / WARN / REJECT
   - **知识链路** 标签：法规引用、规则命中、LLM 是否跳过
5. 填写 [内测清单.md](./内测清单.md)

## 重要声明

- 输出为 **决策辅助**，**不构成法律意见或合规放行**
- 演示法规与规则为 **SG 简化示例**，非正式法源
- 当前使用 **Stub LLM**；长图 / 视频 / OCR **不在范围内**
- 英文版文档见 [README.md](./README.md)（Legal Pilot v1）
