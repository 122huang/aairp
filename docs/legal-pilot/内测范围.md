# 法务内测版 — 范围说明（v1）

**状态：** 内部内测 · 非生产环境 · 非法律意见

---

## 内测目标

让法务 / 合规同事在 **可控演示环境** 中体验：

1. 规则引擎 + 法规引用是否 **可解释、可审计**
2. BLOCKER 时 **跳过 LLM** 的行为是否符合预期
3. PASS / WARN / REJECT 与 **Knowledge Trace** 是否便于复核
4. 收集对 **下一版（鉴权、真 LLM、多市场规则、OCR）** 的需求

---

## 范围内 ✅

| 能力 | 说明 |
|------|------|
| 英文广告文案（text） | 5 个固定 SG 演示场景；可粘贴式扩展需走 API |
| SG 健康 / 食品演示规则 | 禁止 cure 类 BLOCKER；#ad 披露；绝对化 WARN |
| 法规引用 | 规则 finding 关联演示法规库 |
| BLOCKER → 跳过 LLM | 确定性路径，非黑盒 |
| HTML 审核报告 | PASS / WARN / REJECT + 理由 |
| 案例库 | 每次审核自动保存，可浏览 |
| 中文 UI | 界面与流程说明为中文；报告正文仍为英文 |

---

## 范围外 ❌

| 限制 | 替代做法 |
|------|----------|
| 长图 / Banner OCR | 内测仅 text；图片场景需人工提供 ocr_text |
| 视频广告 | 不支持 |
| 多市场生产规则包 | 仅 SG demo；其他国家为 dataset 预留 |
| 生产级 LLM | Stub LLM；open-risk 为演示行为 |
| 法务签批 / 放行 | 系统输出须人工复核 |
| 账号 / 权限 | localhost 开放 API，无 RBAC |
| 外网部署 | 当前为本地 / 内网演示 |

详见 [known-issues.md](../known-issues.md)

## 免责声明

演示法规与规则为 **培训用简化资产**，不得作为正式合规依据。
